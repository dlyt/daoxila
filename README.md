### 搭建Koa服务器
#### node版本
V8.0.0
#### 目录结构
```
.
├── bin                      
│   └── server.js            # 启动文件，引入 babel 和文件`server.js`。
├── src                      # 程序源文件         
│   └── client               # 前端文件夹
│         ├── index.html     # 页面  
│         └── index.js       # 渲染DOM  
├── .babelrc                 # babel配置文件
└── index.js                 # 程序入口文件
```
